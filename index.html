Flowboard

R3F + GSAP test project for Webflow

This is a CodeSandbox that uses Vite for bundling scripts, and also comes with React, ThreeJS and R3F by default. Also added GSAP + ScrollTrigger to play the scroll animation.

Some extras are: GSAP + TextSplit to provide text animations via data attributes on Webflow; Lenis for smooth scroll; and a custom script to listen to specific events and trigger clicks on specific elements on Webflow (using data attributes as weel, this is usefull for interactions).

How to use this:

Clone this template, do your changes, and once you're ready, run the "build" command. It will generate a bundled, static JS file that you can embed directly in Webflow. Grab the URL for the JS file and add on your site using an Embed component or the custom code are on page settings.

Make sure to also include the root div (make sure to use the proper ID and also use data-attributes to include the .GLTF url).
